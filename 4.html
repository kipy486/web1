4. 정보를 잘 정리하는 구조 만들기
html : head,body를 감싸는 단 하나의 최고위층 tag.
!doctype html(이 문서는 html 문서이다.) -- 관용적으로 이렇게 사용함.
head : 본문을 설명. title(본문의 제목), meta charset(본문이 저장된 방식)
1) title : 웹페이지의 제목을 사용자에게 명시적으로 알려줌 / 검색엔진이 표지처럼 이용
2) 글자가 깨지는 것 : 0,1로 저장된 정보가 UTF-8로 저장되었는데 웹브라우저가 열 때 UTF-8로 열지 않은 것
meta charset="utf-8" : 웹브라우저가 문자규칙을 utf-8로 문서를 읽도록 하는 tag
body : 본문
5. a(anchor): link tag. 도시의 길, 인체의 혈관같은 것. 정보의 바다에 정박하는 것. HyperText.
6. 검색: html specification(W3C의 html 공식설명서)
